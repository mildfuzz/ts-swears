# TS Swears

Tiny module to test if a word is a swear (as listed by [Google-profanity-words](https://github.com/RobertJGabriel/Google-profanity-words))

### Methods

```typescript
import isSwear from 'ts-swears'

// isSwear(word: string): boolean
isSwear('fuck') // true
isSwear('flip') // false
```
